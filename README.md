# Resumen del proyecto

**Blockchain básico con Javascript**: En este proyecto trata sobre como se puede desarrollar un pequeño programa de Blockchain usando JavaScript, siguiendo el curso de Proyectos Guiados de Coursera Network

# Objetivos del Proyecto

Este proyecto, sigue **dos objetivos** de aprendizaje:

1. Diseñar un blockchain con Javascript. 
2. Entender los conceptos básicos de un blockchain.
 

Al final de este proyecto, podrás comprender cómo funcionan las cadenas de bloques para luego trabajar sobre ellas.

# Requisitos

El único requisito es tener conocimientos básicos en Javascript.

# Requerimientos

- Node Js
- Libreria: crypto-js

# Estructura del proyecto

El proyecto práctico de **Blockchain básico con Javascript** se divide en las siguientes tareas:

- Inicalizar el entorno: `npm init -y`.
- crear el archivo js con su editor de preferencia.
- ejecutar el codigo, probar y corregir errores.
